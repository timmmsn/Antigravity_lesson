Python 環境建置 (VS Code)

本章節將引導您將 VS Code 打造為專業的 Python 開發環境。

1. 安裝繁體中文語言包

  • 搜尋擴充功能 Chinese (Traditional) 並安裝。
  • 安裝後點擊 Restart 重啟。
  • 若需手動切換：Ctrl+Shift+P > 輸入 "Configure Display Language" > 選 zh-tw。

2. 安裝 Python 擴充套件

  • 搜尋 Python (Microsoft 出品)。
  • 安裝後它會自動配置 Pylance 語法檢查。

3. 建立虛擬環境 (.venv)

在終端機 (Ctrl + `) 輸入：

python -m venv .venv
PowerShell 權限修復：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

4. 安裝專案套件

確認終端機顯示 (.venv) 後，安裝 requirements.txt：

pip install -r requirements.txt

Antigravity Arduino 設定

解決 Antigravity 搜尋不到微軟官方套件的問題 (手動 VSIX 安裝法)。

VSIX 下載連結產生器

連結已產生 下載 .vsix
1. 主程式 (Arduino Maker Workshop)
2. 助手 (Serial Monitor)
3. 修復 C/C++ 二進位檔錯誤

請下載 Windows x64 專用版：

GitHub 下載 win32-x64.vsix

故障排除：第一行紅線 (Unable to handle compilation)

請在 .vscode/ 下建立 c_cpp_properties.json 並貼上：

{
  "configurations": [
    {
      "name": "Arduino",
      "includePath": ["${workspaceFolder}/**"],
      "compilerPath": "", 
      "compilerArgs": [],
      "cStandard": "c17",
      "cppStandard": "c++17"
    }
  ],
  "version": 4
}

專案套件說明

常用的 Python 開發套件，包含安裝與使用說明。

PySide6 GUI 介面開發 (Qt)

Python 的官方 Qt 綁定，用於開發跨平台的現代桌面應用程式。

pip install PySide6
pyserial 硬體序列埠通訊

封裝了 Serial Port 通訊功能，常用於與 Arduino 或工業設備連結。

pip install pyserial
pymysql MySQL 資料庫連線

純 Python 寫的 MySQL 客戶端，輕量且易於部署，適合資料庫操作。

pip install pymysql
PyYAML 讀取 YAML 設定檔

解析 YAML 格式文件，常作為專案的 config.yaml 設定檔讀取器。

pip install PyYAML
numpy / sounddevice 音訊處理與運算

Numpy 處理矩陣運算，Sounddevice 用於即時錄音與播放波形。

pip install numpy sounddevice
auto-py-to-exe 打包成 .exe 執行檔

圖形化介面的 PyInstaller 封裝工具，將 Python 腳本轉為 Windows 執行檔。

pip install auto-py-to-exe

PySide6 UI 設計流程

1. 啟動 Designer
pyside6-designer
2. 轉換為 Python (.ui -> .py)
pyside6-uic ui/main.ui -o ui/ui_main.py
3. 主程式呼叫 
from ui.ui_main import Ui_MainWindow
...
self.ui = Ui_MainWindow()
self.ui.setupUi(self)

Git 版本控制與回溯

在 Antigravity 中輕鬆管理程式碼版本，隨時時光倒流。

1. 初始化 Git (Init)

在專案資料夾開啟終端機，建立新的數據庫：

git init

2. 提交變更 (Commit)

將目前的進度存檔：

# 加入所有檔案
git add .
# 提交並說明
git commit -m "完成功能開發"

版本回溯 (Reset / Checkout)

查看歷史紀錄 git log --oneline
回到過去 (不保留變更)

危險！此操作會遺失目前進度。

git reset --hard <commit_id>
切換版本 (唯讀檢視) git checkout <commit_id>

Skill Seekers 教學

將官方文件轉化為 AI 知識庫的神器。

1. 抓取 (Scrape)

skill-seekers scrape --name pyyaml --url https://pyyaml.org/wiki/PyYAMLDocumentation

2. 打包 (Package)

skill-seekers package output/pyyaml/
3. 上傳給 AI： 將產生的 .zip 檔傳給 Antigravity，AI 就能學會該套件用法。

Anthropic Skills (MCP) 應用場景指南

這些 "Skills" (現稱為 MCP Servers) 是讓 AI 連接真實世界的橋樑。
以下是 GitHub 官方庫中主要技能的實際落地應用分析。

Filesystem

系統操作

讓 AI 能夠讀取、寫入、移動和列出您電腦或伺服器上的檔案。

🎯 應用場景

1. 自動化程式開發 (Coding Agent) AI 不只是給建議，而是直接幫您修改專案中的 .py 或 .js 檔案，甚至幫您寫好 README。
2. 記錄檔分析與清理 讓 AI 讀取 /logs 資料夾，分析錯誤模式，並自動歸檔舊的日誌文件。

PostgreSQL

數據管理

賦予 AI 唯讀或讀寫權限來存取關聯式資料庫，執行 SQL 查詢。

🎯 應用場景

1. BI 商業智慧分析師 老闆問：「上個月銷售額最高的產品是？」AI 自動轉成 SQL 查詢庫存與訂單表，並回傳統計結果。
2. 自動化運維 (DBA 助手) 讓 AI 檢查資料庫 Schema，建議索引優化，或是檢查數據異常 (如重複用戶)。

Brave Search

網路資訊

讓 AI 能夠聯網搜尋最新的網路資訊 (比訓練數據更新)。

🎯 應用場景

1. 市場競品調研 AI 自動搜尋「2025年 AI 趨勢」，彙整多篇文章摘要，生成最新的市場報告。
2. 技術文件除錯 當遇到新版 Library 報錯時，AI 上網搜尋最新的 GitHub Issues 或 StackOverflow 解決方案。

Puppeteer

瀏覽器自動化

控制無頭瀏覽器 (Headless Browser)，執行網頁操作、截圖、或是爬取動態內容。

🎯 應用場景

1. 網站自動化測試 AI 自動開啟您的網站，點擊按鈕，填寫表單，確認流程是否正常，並在報錯時截圖。
2. 動態網頁爬蟲 抓取需要登入或 JavaScript 渲染後的網頁資料 (例如抓取股票即時走勢圖)。

Google Maps

地理資訊

存取地點資訊、導航路線、查詢周邊設施。

🎯 應用場景

1. 差旅/旅遊規劃助手 「幫我規劃台北到高雄的行程，包含中途評價 4.5 星以上的咖啡廳。」
2. 物流路徑優化 計算多個地點之間的距離與時間，協助安排送貨順序。

Bash / Command

系統操作

讓 AI 直接在終端機執行指令 (需極度小心權限)。

🎯 應用場景

1. 環境建置與部署 「幫我安裝 Python 3.10 並建立虛擬環境，然後安裝 requirements.txt。」
2. 系統監控與修復 執行 top 查看 CPU 佔用，或是重啟當機的服務 (systemctl restart)。

什麼是 MCP (Model Context Protocol)？

您在該 GitHub 看到的其實是 MCP Servers 的範例。
以前我們要讓 AI 使用工具，需要寫很多複雜的 Python/API 串接程式碼。 現在透過 MCP 協定，您只要啟動這些 "Server" (例如啟動 Filesystem Server)， Claude Desktop App 或任何支援 MCP 的 AI (如 Antigravity 內建的 Agent) 就能自動發現並使用這些工具，完全不需要額外寫程式串接。
